Notes¶

To get uwsgi running I basically followed Nginx, Flask, uWSGI integration except when it came time to run it automatically with systemctl. Since I’d already installed supervisor to run jupyter (Deploying Jupyter in Ubuntu with Nginx and Supervisor I did something similar with uwsgi:

uwsgi.conf in /etc/supervisor/conf.d:

[program:uwsgi]
command = uwsgi --ini myproject.ini
user=root
directory = /srv
autostart = true
autorestart = true
stdout_logfile = /var/log/jupyter/uwsgi.log
redirect_stderr = true

this works in conjunction with /srv/myproject.ini:

[program:uwsgi]
command = uwsgi --ini myproject.ini
user=root
directory = /srv
autostart = true
autorestart = true
